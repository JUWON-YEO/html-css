###### EUID ╳ Likelion

## Remote 이름 변경하기

```sh
git remote rename origin upstream
```

## Remote 추가하기

```sh
git remote add origin https://github.com/<본인의 Github 계정 ID>/basic-html-css.git
```

## Remote 조회하기

```sh
git remote -v
```

upstream의 teacher 브랜치의 업데이트 된 내용을 가져오기

```sh
git pull upstream teacher
```
